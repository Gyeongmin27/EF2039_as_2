# 동체시력 테스트 웹 게임

동체시력을 테스트하고 향상시키기 위한 교육용 웹 게임입니다.

## 프로젝트 개요

이 프로젝트는 "Introduction to AI Programming" 과목의 Assignment 3으로, PRD 문서 작성과 웹사이트 구현을 포함합니다.

## 기능

- **랜덤 단어 생성**: 2~3글자의 한글 단어를 랜덤으로 선택
- **난이도 설정**: 글자 획수에 따라 상, 중, 하 난이도 선택 가능
- **글자 떨어뜨리기**: 화면 위쪽의 중앙 주변 랜덤 위치에서 글자가 떨어짐 (랜덤 순서)
- **속도 조절**: 글자가 떨어지는 속도를 조절할 수 있음
- **시간 제한**: 글자가 모두 떨어진 후 1.5초 이내에 답 입력
- **점수 시스템**: 정답 시 점수 증가 및 자동 다음 라운드 진행, 오답 시 획득한 점수 표시 후 게임 종료

## 기술 스택

- **HTML5**: 웹 페이지 구조
- **CSS3**: 스타일링 및 애니메이션
- **JavaScript (ES6+)**: 게임 로직 및 인터랙션

## 파일 구조

```
as_3/
├── index.html          # 메인 게임 페이지
├── settings.html       # 설정 페이지
├── about.html          # 게임 소개 페이지
├── styles.css          # 공통 스타일시트

├── words.js            # 한글 단어 데이터베이스
├── PRD.md              # 제품 요구사항 문서
└── README.md           # 프로젝트 설명 문서
```

## 사용 방법

1. `index.html` 파일을 웹 브라우저에서 엽니다.
2. 설정 페이지에서 난이도와 속도를 조절할 수 있습니다.
3. "게임 시작" 버튼을 클릭하여 게임을 시작합니다.
4. 떨어지는 글자들을 관찰하고 기억합니다 (글자들은 랜덤 순서로 떨어집니다).
5. 모든 글자가 떨어진 후 입력창에 단어를 입력합니다.
6. 1.5초 이내에 정답을 입력하면 점수가 올라가고 자동으로 다음 라운드가 시작됩니다.
7. 오답을 입력하면 획득한 점수가 표시되고 게임이 종료됩니다.

## 게임 규칙

- 각 라운드마다 2~3글자의 한글 단어가 랜덤으로 선택됩니다.
- 글자는 화면 위쪽의 중앙 주변 랜덤 위치에서 떨어집니다 (가장자리 제외).
- 글자들은 랜덤 순서로 떨어집니다.
- 모든 글자가 떨어진 후 1.5초 이내에 답을 입력해야 합니다.
- 정답 입력 시 난이도에 따라 점수가 증가하고 자동으로 다음 라운드가 시작됩니다:
  - 하: 10점
  - 중: 20점
  - 상: 30점
- 오답 입력 시 획득한 점수가 표시되고 게임이 종료되어 처음 화면으로 돌아갑니다.

## 난이도 설명

- **하**: 획수 4개 이하의 간단한 글자
- **중**: 획수 5~9개의 중간 복잡도 글자
- **상**: 획수 10개 이상의 복잡한 글자
